Sodapop: A Coding Challenge

SodaPop is an application based around a coding challenge in which a "soda" value and "pop" value are provided by the user. The result comes from a loop in which the value for each as well as all multiples of that value are replaced with the word "soda" or "pop" respectively. Values that are multiples of both are replaced with the word "sodapop".

Instructions

Enter a value for "soda" and a value for "pop" and click the "SodaPop" button. The results table with the words replacing their respective values and multiples will then appear.
